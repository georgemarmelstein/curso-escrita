# 0.5 Introdução ao Modo Agêntico

## Objetivo
Entender a mudança de paradigma dos LLMs — de assistente que responde para agente que usa ferramentas.

## A Mudança de Paradigma

Os LLMs evoluíram. Antes, funcionavam assim:

```
MODELO ANTIGO (Assistente)

   Você pergunta ──► Modelo responde

   Fim.
```

Agora, funcionam assim:

```
MODELO ATUAL (Agente)

   Você instrui ──► Modelo DECIDE se precisa de ferramentas
                         │
                         ├── Precisa buscar na web? → Busca
                         ├── Precisa calcular? → Calcula
                         ├── Precisa criar documento? → Cria artefato
                         └── Não precisa? → Responde direto

   Modelo entrega resultado (com ou sem ferramentas)
```

A diferença fundamental: **o modelo decide por conta própria quando usar ferramentas**. Você pode direcionar, mas ele também ativa automaticamente quando julga necessário.

## O que são Ferramentas?

No modo agêntico, o Claude tem acesso a capacidades extras:

| Ferramenta | O que faz | Quando ativa |
|------------|-----------|--------------|
| **Busca web** | Pesquisa informações atualizadas | Perguntas sobre eventos recentes, normas, jurisprudência |
| **Artefatos** | Cria documentos, tabelas, código | Pedidos de criação estruturada |
| **Cálculo** | Executa operações matemáticas | Contas complexas |
| **Análise de dados** | Processa planilhas, extrai padrões | Documentos com dados numéricos |
| **Pensamento prolongado** | Raciocínio mais profundo | Problemas complexos |

## Vantagem do Claude: Transparência

O Claude **sempre informa** quando está usando uma ferramenta. Você vê mensagens como:

- "Buscando na web..."
- "Criando artefato..."
- "Usando ferramenta de análise..."

**Isso é importante.** Outros modelos (ChatGPT, Gemini) também usam ferramentas automaticamente, mas nem sempre informam o que estão fazendo. Você recebe a resposta sem saber se veio do conhecimento interno ou de uma busca externa.

No Claude, a transparência permite que você:
- Saiba a origem da informação
- Decida se quer ativar ou desativar ferramentas
- Entenda o processo de raciocínio do modelo

## Exercício: Entendendo o Modo Agêntico

### O que você vai fazer
Experimentar as ferramentas do Claude e observar quando ele as ativa automaticamente.

### Tarefa 1: Busca Ativada

**Passo a passo:**
1. Abra uma nova conversa no Claude
2. **Ative a ferramenta de busca** (clique no ícone de busca ou globo)
3. Pergunte: `Explique as condições de uso dos LLMs segundo a Resolução 615 do CNJ.`

**O que observar:**
- Como o Claude informa que está buscando
- A diferença entre conhecimento interno e busca em tempo real
- Como ele cita e referencia as fontes encontradas
- A qualidade e atualização das informações

### Tarefa 2: Busca Desativada

**Passo a passo:**
1. **Abra uma nova conversa** (importante: limpar contexto)
2. **Desative a ferramenta de busca**
3. Faça a mesma pergunta: `Explique as condições de uso dos LLMs segundo a Resolução 615 do CNJ.`

**O que observar:**
- O que acontece quando o modelo não pode buscar
- Compare com a resposta da Tarefa 1
- Note as diferenças de precisão e atualização
- Observe se o modelo reconhece suas limitações (data de corte)

### Tarefa 3: Artefatos

**Passo a passo:**
1. Abra uma nova conversa
2. Peça: `Crie uma tabela com os conceitos fundamentais para entender os LLMs.`

**O que observar:**
- Se o Claude ativa o artefato automaticamente
- A organização e estruturação das informações
- Como o artefato pode ser editado e refinado
- A capacidade de síntese de conceitos complexos

### Tarefa 4: Cálculo

**Passo a passo:**
1. Abra uma nova conversa
2. Pergunte: `Quanto é 9989984719 x 70989808013?`
3. Observe se ele ativa alguma ferramenta de cálculo
4. Se não ativar, peça explicitamente para usar ferramenta de cálculo
5. Compare os resultados

**O que observar:**
- Se o modelo ativa automaticamente ferramentas de cálculo
- A diferença entre "cálculo mental" (aproximado) e cálculo preciso
- Como o modo agêntico reconhece quando precisa de ferramentas
- A precisão dos resultados com e sem ferramentas

### Checklist de Verificação
- [ ] Testei a busca web ativada
- [ ] Testei a busca web desativada e comparei
- [ ] Vi o Claude criar um artefato
- [ ] Testei cálculo com e sem ferramenta
- [ ] Entendo que o Claude decide automaticamente quando usar ferramentas
- [ ] Percebi que o Claude informa quando está usando ferramentas

## Conceito-Chave

O **modo agêntico** permite que os LLMs usem ferramentas externas **automaticamente** (busca web, cálculos, criação de artefatos, análise de dados) para fornecer respostas mais precisas e atualizadas.

O modelo não espera você pedir — ele **decide** quando precisa de ajuda externa. Isso é fundamentalmente diferente do modelo "pergunta → resposta" tradicional.

## Dica Prática
> Observe as mensagens do Claude durante o processamento. Quando você vê "Buscando..." ou "Criando artefato...", está vendo o modo agêntico em ação. Essa transparência é exclusiva do Claude e ajuda você a entender de onde vem a informação.

## Armadilha Comum
> **"Vou sempre ativar todas as ferramentas para ter respostas melhores."**
>
> Não funciona assim. Cada ferramenta tem um propósito:
> - Busca web → informações atualizadas (mas pode trazer ruído)
> - Artefatos → documentos estruturados (mas nem tudo precisa ser artefato)
> - Cálculo → precisão matemática (mas o modelo sabe quando precisa)
>
> Deixe o Claude decidir quando usar. Ele foi treinado para isso. Intervenha apenas quando quiser forçar ou impedir uma ferramenta específica.
