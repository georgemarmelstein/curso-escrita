# 0.1 Primeiros Passos

## Objetivo
Criar sua conta no Claude e escolher o plano adequado para uso profissional no Direito.

## Por que Claude como Base?

Neste curso, usamos o **Claude** (da Anthropic) como plataforma principal por três razões:

| Razão | O que significa |
|-------|-----------------|
| **Robustez cognitiva** | Projetado para atividades de alta profundidade analítica — ideal para análise jurídica, argumentação complexa e escrita técnica |
| **Transparência** | A Anthropic explica como o modelo funciona e dá ao usuário controle sobre seus dados |
| **Didática** | Interface limpa, respostas estruturadas, comportamento previsível — facilita o aprendizado |

Os conceitos que você aprenderá aqui se aplicam a qualquer LLM (ChatGPT, Gemini, etc.), mas usaremos Claude para demonstrações e exercícios.

## Os Planos do Claude

| Plano | Preço | Para quem | Uso no Direito |
|-------|-------|-----------|----------------|
| **Free** | R$ 0 | Curiosos, testes iniciais | Insuficiente para trabalho real |
| **Pro** | ~US$ 17-20/mês | Profissionais, uso diário | **RECOMENDADO** |
| **Max** | US$ 100-200/mês | Uso massivo, em escala | Para alta demanda |

### Pro (Recomendado para Direito)
- 5x mais capacidade que Free
- Acesso a todos os modelos (incluindo Opus)
- Extended thinking (raciocínio profundo)
- Prioridade em horários de pico

### Max (Para Usuários Avançados)
- 5x a 20x mais capacidade que Pro
- Acesso prioritário a novos recursos
- Ideal para uso em escala ou escritórios com alta demanda

## Exercício: Criar Conta e Escolher Plano

### O que você vai fazer
Criar sua conta no Claude e, se for usar profissionalmente, assinar o plano Pro.

### Passo a passo

**1. Acessar o site:**
- Abra o navegador (Chrome recomendado)
- Acesse: **https://claude.ai**

**2. Criar conta:**
- Clique em **"Sign up"** ou **"Criar conta"**
- Escolha método de autenticação:
  - Email + senha, OU
  - Conta Google, OU
  - Conta Apple
- Complete a verificação de email

**3. Primeiro acesso:**
- Aceite os termos de uso
- Você entrará no plano Free automaticamente

**4. Upgrade para Pro (recomendado):**
- Clique no seu perfil (canto inferior esquerdo)
- Vá em **"Ver todos os planos"**
- Selecione **"Pro"**
- Insira dados de pagamento
- Confirme a assinatura

### Checklist de Verificação
- [ ] Conta criada no claude.ai
- [ ] Email verificado
- [ ] Consegui acessar a interface de chat
- [ ] (Se Pro) Assinatura ativa confirmada
- [ ] Entendo a diferença entre Free, Pro e Max

## Exemplo Resolvido

### Situação
Magistrado que vai usar Claude para análise de processos diariamente.

### Decisão
```
Perfil: Uso profissional, ~10-30 análises/dia, documentos complexos
Plano escolhido: Pro (US$ 17-20/mês)

Justificativa:
- Free seria insuficiente (limite baixo, sem prioridade)
- Pro oferece capacidade adequada + extended thinking
- Max seria excessivo para uso individual
```

## Dica Prática
> Comece no Free para testar por alguns dias. Quando perceber que está "batendo no limite" frequentemente, faça o upgrade para Pro. Assim você sente a diferença na prática.

## Armadilha Comum
> **"Vou usar o Free porque não quero gastar."**
>
> Para uso profissional no Judiciário, o Free é inadequado:
> - Limites baixos interrompem o trabalho
> - Sem prioridade = lentidão em horário comercial
> - Sem extended thinking = análises menos profundas
>
> O Pro custa menos que um livro jurídico por mês.
