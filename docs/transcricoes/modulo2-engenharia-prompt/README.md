# Módulo 2 - Engenharia de Prompt

**Curso:** Escrita com LLMs
**Autor:** George Marmelstein
**Total de Aulas:** 6

---

## Visão Geral

Este módulo desenvolve competências práticas e teóricas em engenharia de prompt, capacitando o aluno a comunicar eficientemente com LLMs e extrair seu potencial máximo.

**Definição Central:** Engenharia de Prompt é a disciplina que estuda como estruturar entradas (prompts) para LLMs de forma a obter saídas desejadas de maneira confiável e eficiente.

## Três Modalidades de Prompt

```
VIBE PROMPTING
├── Linguagem natural pura
├── Conversacional e direto
└── Fluxo de pensamento

PROMPT ESTRUTURADO
├── Markdown e delimitadores
├── Frameworks (P.O.E.M.A.)
└── Controle de output

PROMPT AGÊNTICO
├── Orquestração de tarefas
├── Tools, skills, resources
└── Workflows complexos
```

## Conceitos-Chave por Aula

### Aula 1 - Introdução
- Linguagem natural como programação
- Três modalidades de prompt
- Vibe, Estruturado, Agêntico

### Aula 2 - Fundamentos (Parte 1)
- LLM como máquina de calcular textos
- LLM como máquina obediente
- Dirigibilidade (Steerability)
- Bajulação (Sycophancy)

### Aula 3 - Fundamentos (Parte 2)
- Inputs Nobres = Outputs Nobres
- Few-shot Learning
- Chain of Thought
- Pensamento Estendido
- Efeito Priming

### Aula 4 - Vibe Prompting
- Solicitações polidas vs comandos
- Prompts emocionais
- Estímulos positivos e negativos
- Personas e POV

### Aula 5 - Os Três Desafios
- **Dirigibilidade:** clareza, estrutura, CoT
- **Qualidade do Conhecimento:** RAG, curadoria, humildade
- **Qualidade do Estilo:** formatação, personas, priming

### Aula 6 - Prompt de Sistema
- Hierarquia de prompts (sistema, customização, usuário)
- Contrato de comportamento
- Análise do prompt do Claude
