## Slide 2 - O LLM como uma Máquina de Calcular Textos

**O LLM COMO UMA MÁQUINA DE CALCULAR TEXTOS**

- **INPUT → CÁLCULOS → OUTPUT** (não-determinístico)
- **PROCESSO AUTO-REGRESSIVO** (uma palavra de cada vez)
- **PADRÕES LINGUÍSTICOS** (probabilidade semântica)
- **SEM APRENDIZADO CONTÍNUO** (reset de contexto)
- **APRENDIZADO ATRAVÉS DO CONTEXTO**
- **LIMITADO E GUIADO PELA JANELA DE CONTEXTO**

## Slide 3 - O LLM como uma Máquina Obediente

**O LLM COMO UMA MÁQUINA OBEDIENTE**

- **PRÉ-TREINO E FINE TUNING** (feedback humano)
- **ALINHAMENTO ÉTICO** (HHH)
- **DIRIGIBILIDADE** (Steerability)
- **EFEITO DE BAJULAÇÃO** (Sycophancy)
- **ESPELHAMENTO DE INTELIGÊNCIA**
- **SEM SABEDORIA INTRÍNSECA. SEM SENSO DE JUSTIÇA**
