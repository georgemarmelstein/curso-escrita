Slash Commands

// Claude Code Reference
?/help
Mostra ajuda e lista todos os comandos disponiveis
↺/clear
Limpa historico da conversa (fresh start)
▼/compact
Comprime conversa para economizar tokens
★/init
Inicializa CLAUDE.md para o projeto
✕/exit
Sai da sessao Claude Code
$/cost
Mostra custos de tokens da sessao atual
█/context
Visualiza uso de contexto (fuel gauge)
●/status
Versao e status de conectividade
+/doctor
Executa diagnosticos de instalacao
⚙/config
Abre menu de configuracoes gerais
🔒/permissions
Gerencia permissoes de ferramentas
⍑/model
Troca modelo (Sonnet, Opus, Haiku)
🗒/memory
Abre CLAUDE.md para edicao
✎/vim
Ativa modo vim para edicao
▶/resume
Retoma uma sessao anterior
📁/add-dir
Adiciona diretorios de trabalho
📋/plan
Ativa modo planejamento
🤖/agents
Gerencia subagentes customizados
💡/skills
Lista skills disponiveis no projeto
🔗/hooks
Mostra hooks configurados
claude -- ~/projeto
