/clear vs /compact

Clique nos botões para simular os comandos

/clear
Limpa completamente a conversa
Contexto Antes 50K tokens
Contexto Depois 0 tokens
O que acontece
✓ CLAUDE.md é re-injetado
✗ Histórico da conversa: PERDIDO
✓ Arquivos criados: mantidos
✓ Configurações: mantidas
Quando usar
  • Nova tarefa, contexto diferente
  • Qualidade das respostas caiu
  • Contexto está confuso
  • Começar do zero
/compact
Resume o contexto preservando o essencial
Contexto Antes 50K tokens
Contexto Depois ~10K tokens
O que acontece
✓ CLAUDE.md é re-injetado
✓ Decisões críticas: PRESERVADAS
~ Detalhes: resumidos
✓ Contexto essencial: mantido
Quando usar
  • Tarefa longa, mesma área
  • Preservar contexto essencial
  • Contexto quase cheio (80%+)
  • Continuar mesma sessão
