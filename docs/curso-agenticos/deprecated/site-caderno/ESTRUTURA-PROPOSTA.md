## Conceitos-Chave por Aula

### Aula 0: Bem-Vindo ao Mundo Agêntico
- Mudança de paradigma: operador → regente
- Inversão 80/20 (humano como middleware)
- Por que o terminal?

### Aula 1: A Era dos Agentes
- Assistente vs Protoagente vs Agente
- Escala de autonomia L1-L5
- Os 3 pilares (Michele Catasta)
- Paradoxo METR

### Aula 2: Desvendando o Claude Code
- Instalação (npm, brew, binário)
- Anatomia da interface
- Comandos essenciais (/help, /status, /cost)
- CLAUDE.md (instruções de projeto)
- Modo YOLO e permissões

### Aula 3: Arquitetura de Agentes
- Pipeline vs Workflow vs Agent Loop
- Gates (validação)
- Routers (decisão)
- Smart Zone vs Dumb Zone (12 Factor)
- Gerenciamento de contexto

### Aula 3.5: Agent Teams
- Subagentes especializados
- Task Tool e tipos de agente
- Paralelização
- Isolamento de contexto

### Aula 4: Construindo os Primeiros Agentes
- Anatomia de uma Skill (SKILL.md)
- Anatomia de um Command
- Pattern RPI (Research-Plan-Implement)
- Feature-dev workflow
- Debugging e iteração

### Aula 4.5: Hooks — Automação Determinística
- O que são Hooks
- Eventos disponíveis
- Casos de uso jurídicos
- Combinando com Skills

### Aula 5: Sistema Jurídico Agêntico (CLERK)
- MCPs jurídicos (BNP, JULIA, CJF)
- Integração PJE
- Pipeline: Baixar → Converter → Relatar → Analisar → Minutar
- Revisão com subagentes paralelos

### Aula 6: Engenharia de Prompt
- Estrutura de prompts eficazes
- Técnicas: few-shot, chain-of-thought, self-consistency
- Prompts jurídicos específicos

### Aula 6B: Prompts em Sistemas Agênticos
- Diferenças para prompts tradicionais
- SEMPRE/NUNCA/PREFERIR
- Instruções para agentes autônomos

### Aula 7: Engenharia de Contexto
- CLAUDE.md avançado
- Memória e compactação (/compact)
- Gerenciamento de tokens
- Progressão de instrução

### Aula 8: Engenharia de Capacidades
- MCP (Model Context Protocol)
- Criando ferramentas personalizadas
- Recursos e templates

### Aula 9: Engenharia de Workflows
- Orquestração de pipelines
- Fluxos condicionais
- Tratamento de erros
- Padrões de produção

### Aula 10: Projeto Integrador
- Especificação do projeto
- Implementação guiada
- Revisão e ajustes
- Publicação
