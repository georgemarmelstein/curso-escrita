Kit de Instalacao

Requisitos Minimos

Antes de instalar o Claude Code, certifique-se de ter os requisitos minimos do sistema.

Requisitos do Sistema

Componente Requisito
Sistema Operacional Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+)
Node.js Versao 18 ou superior
npm Versao 9 ou superior
Memoria RAM 8 GB (recomendado 16 GB)
Espaco em Disco 500 MB para instalacao

Instalacao Rapida

Metodo Recomendado

A forma mais simples de instalar o Claude Code e via npm global.

# Instalar Claude Code globalmente
npm install -g @anthropic-ai/claude-code

# Verificar instalacao
claude --version

# Autenticar (sera solicitada a chave API)
claude auth

# Testar
claude "Ola, Claude!"

Instalacao Alternativa (Windows)

# Via PowerShell (Administrador)
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
npm install -g @anthropic-ai/claude-code

# Se houver erro de permissao
npm config set prefix %USERPROFILE%\.npm-global
$env:PATH += ";$env:USERPROFILE\.npm-global"

Verificacao da Instalacao

# Verificar versao
claude --version

# Verificar autenticacao
claude auth status

# Verificar MCPs configurados
claude /mcp

# Executar diagnostico
claude /doctor
Problemas Comuns
  • Erro de permissao: Execute como administrador ou configure prefix do npm
  • Node nao encontrado: Reinstale o Node.js e reinicie o terminal
  • Erro de autenticacao: Verifique a chave API em ~/.claude/settings.json

MCPs Juridicos

Sobre os MCPs do Curso

Estes MCPs foram desenvolvidos especificamente para o contexto juridico brasileiro, permitindo pesquisa de precedentes e jurisprudencia diretamente do Claude Code.

BNP-API (Banco Nacional de Precedentes)

┌─────────────────────────────────────────────────────────────┐
│                      MCP: BNP-API                           │
├─────────────────────────────────────────────────────────────┤
│  FONTE: Banco Nacional de Precedentes (CNJ/PANGEA)          │
│                                                             │
│  COBERTURA:                                                 │
│  ├── Repercussao Geral (STF)                                │
│  ├── Recursos Repetitivos (STJ)                             │
│  ├── Sumulas Vinculantes                                    │
│  └── IRDRs e IACs                                           │
│                                                             │
│  TOOLS:                                                     │
│  ├── buscar_precedentes    → Busca estruturada              │
│  └── gerar_relatorio       → Relatorio formatado            │
│                                                             │
│  SINTAXE: +termo -exclusao "frase exata"                    │
└─────────────────────────────────────────────────────────────┘

Exemplos de Uso

# Busca simples
"Pesquise precedentes sobre BPC para idoso estrangeiro"

# Busca estruturada
+aposentadoria +especial +EPI

# Tema especifico
"tema 1066"

Instalacao dos MCPs

# 1. Clonar repositorio dos MCPs
git clone https://github.com/seu-repo/mcps-juridicos.git

# 2. Instalar dependencias
cd mcps-juridicos
pip install -r requirements.txt

# 3. Configurar em ~/.claude/settings.json
{
  "mcpServers": {
    "bnp-api": {
      "command": "python",
      "args": ["caminho/para/bnp_server.py"]
    },
    "cjf-jurisprudencia": {
      "command": "python",
      "args": ["caminho/para/cjf_server.py"]
    },
    "julia-trf5": {
      "command": "python",
      "args": ["caminho/para/julia_server.py"]
    }
  }
}

# 4. Reiniciar Claude Code e verificar
claude /mcp

Boas Praticas da Comunidade

  • Compartilhe aprendizados: Documente o que funciona (e o que nao funciona)
  • Contribua com MCPs: A comunidade se beneficia de novas integracoes
  • Reporte bugs: Issues bem documentadas ajudam a melhorar o produto

Dica para Juristas

Considere criar um grupo de discussao especifico para uso de IA no Judiciario. A troca de experiencias entre colegas acelera o aprendizado e identifica casos de uso relevantes.
