# Curso de Sistemas Agênticos no Direito

**Trilha Avançada** do Caderno de Exercícios — Formação em IA para o Direito.

## Pré-requisito

Este curso pressupõe conhecimento do curso base **Formação em IA no Direito** (Trilha Fundamentos).

## Estrutura de Aulas

| Aula | Tema |
|------|------|
| A0 | Introdução a Sistemas Agênticos |
| A1 | Arquitetura de Agentes |
| A2 | Ferramentas e Tool Use |
| A3 | Orquestração de Agentes |
| A3.5 | Padrões de Orquestração |
| A4 | Memória e Contexto |
| A4.5 | Estratégias de Memória |
| A5 | Loops e Feedback |
| A6 | Agentes Jurídicos na Prática |
| A6b | Casos de Uso Avançados |
| A7 | Segurança e Guardrails |
| A8 | Avaliação de Agentes |
| A9 | Deploy e Produção |
| A10 | Projeto Final |

## Conteúdo Fonte

## Integração com o Caderno

As aulas serão integradas ao Caderno de Exercícios (`site/index.html`) com prefixo `A`:
- Trilha Fundamentos: `1.1`, `1.2`, `2.1`...
- Trilha Avançada (Agênticos): `A1`, `A2`, `A3`...

## Produtos

1. **Slides Web** — Apresentações interativas em `site/slides-agenticos/`
2. **Caderno de Exercícios** — Integrado ao caderno principal com seletor de trilha
