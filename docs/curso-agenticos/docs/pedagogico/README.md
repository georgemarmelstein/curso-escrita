## Visão Geral

Este curso foi construído para ser o **maior, mais profundo, detalhado e prático** programa de formação em sistemas agênticos já desenvolvido em língua portuguesa.

A documentação pedagógica estabelece:
- A **filosofia** que fundamenta o curso
- As **fontes** de conhecimento de ponta utilizadas
- A **metodologia** de construção
- As **competências** desenvolvidas pelo aluno
- As **referências** bibliográficas e citações

## Princípios Fundamentais

### 1. Conhecimento de Ponta, Não Vulgarização

A base do curso são fontes SOTA (State of the Art):
- AI Engineer Summit 2024-2025
- Documentação oficial Anthropic
- Práticas de produção de empresas líderes

### 2. Aumentação, Não Substituição

> *"AI cannot replace thinking. It can only amplify the thinking you have done."*
> — Dex Horthy

O curso forma profissionais que **multiplicam** sua capacidade, não que delegam seu julgamento.

### 3. Meta-Demonstração

O processo de construção do curso **demonstra** a filosofia ensinada:
- Autor como Regente
- Claude Code como Orquestra
- Subagentes para pesquisa paralela
- Iteração com feedback específico

### 4. Prática Deliberada

Cada competência tem exercício de validação. Teoria serve à aplicação.

## Público-Alvo

**Juristas técnicos** (magistrados, advogados, servidores) que querem:
- Dominar sistemas agênticos
- Multiplicar sua capacidade de trabalho
- Manter controle e responsabilidade sobre decisões
