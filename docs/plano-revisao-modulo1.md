## Processo por Aula

Para cada aula, executar:

### 1. Leitura Comparativa
- [ ] Ler transcrição completa
- [ ] Ler aula atual no caderno.html
- [ ] Identificar divergências de conteúdo

### 2. Revisão de Conteúdo
- [ ] Adequar teoria às transcrições
- [ ] Atualizar terminologia
- [ ] Corrigir conceitos desalinhados

### 3. Revisão de Exercícios
- [ ] Comparar com Livro de Exercícios - 2025.txt
- [ ] Verificar se exercício reflete o ensinado na aula
- [ ] Ajustar passos e checklist conforme necessário

### 4. Validação
- [ ] Testar no navegador
- [ ] Conferir formatação

## Ordem de Execução

```
Aula 1.1 → Aula 1.2 → ... → Aula 1.13 (ou 1.16)
```

Revisar uma aula por vez, na ordem, para manter consistência.
