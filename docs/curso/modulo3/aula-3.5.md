## Base Conceitual

### Quem é Susan Haack?

Filósofa americana, especialista em epistemologia, criou o foundherentismo:
- Combina fundacionismo (base sólida) com coerentismo (mútuo suporte)
- Propôs metáfora das palavras cruzadas para justificação
- Aplicou à epistemologia jurídica

```
┌─────────────────────────────────────────────────────────────┐

```

### Critérios de Avaliação (Haack)

| Critério | Pergunta | Aplicação |
|----------|----------|-----------|
| **Suporte** | A evidência suporta a conclusão? | Força probante |
| **Coerência** | A evidência é coerente com as demais? | Encaixe mútuo |
| **Ancoragem** | A evidência tem base independente? | Origem verificável |
| **Compreensividade** | Considera todas as evidências? | Completude |
| **Segurança** | Quão confiável é a cadeia? | Grau de certeza |

### Grau de Justificação

```
GRAU = f(suporte, coerência, ancoragem, compreensividade)

Alto:   Múltiplas evidências independentes, coerentes entre si
Médio:  Algumas evidências, coerência parcial
Baixo:  Poucas evidências, contradições
Nulo:   Contradição fundamental ou ausência de ancoragem
```

## Exercício Prático

### Tarefa
Aplicar framework Haack em conjunto probatório.

### Caso
```
ACUSAÇÃO: Roubo em residência

EVIDÊNCIAS:
1. Testemunha visual: "Vi o acusado entrando na casa às 22h"
2. Câmera de segurança: Mostra pessoa encapuzada entrando às 22:15
3. Alibi do acusado: "Estava em festa com amigos às 22h"
4. Testemunha do alibi: "Acusado chegou na festa às 21h"
5. Digital do acusado: Encontrada no vidro da janela
6. Histórico: Acusado tem antecedentes por furto
7. Reconhecimento fotográfico: Testemunha identificou acusado
```

### Passo a Passo

**Prompt Haack:**

```markdown
# ANÁLISE FOUNDHERENTISTA (HAACK)

## Persona
Você é um analista probatório que aplica o framework
foundherentista de Susan Haack para avaliar conjuntos
de evidências.

## Tarefa
Analise o conjunto probatório abaixo usando a metáfora
das palavras cruzadas de Haack.

## Metodologia

### ETAPA 1: INVENTÁRIO DE EVIDÊNCIAS
Liste cada evidência com:
- Tipo (direta/indireta)
- Fonte
- O que suporta

### ETAPA 2: MATRIZ DE COERÊNCIA
Para cada par de evidências, avalie:
- Coerentes (suportam mesma narrativa)
- Neutras (sem relação)
- Conflitantes (contradizem)

### ETAPA 3: ANCORAGEM
Para cada evidência, avalie:
- Tem base independente?
- Origem é verificável?
- Pode ser fabricada?

### ETAPA 4: PALAVRAS CRUZADAS
Representação visual:
- Quais evidências "encaixam"?
- Quais são problemáticas?
- Há "letras" que não combinam?

### ETAPA 5: GRAU DE JUSTIFICAÇÃO
Conclusão com grau (Alto/Médio/Baixo/Nulo)

## ETAPA 5: GRAU DE JUSTIFICAÇÃO

**Para condenação:** MÉDIO
- 4 evidências coerentes (testemunha, câmera, digital, reconhecimento)
- MAS: alibi com testemunha cria contradição fundamental
- Necessário resolver: alibi vs evidências de presença

**Pontos problemáticos:**
1. Testemunha visual vs testemunha de alibi (horário)
2. Câmera não identifica pessoa (encapuzada)
3. Digital pode ter explicação alternativa

**Recomendação:**
Investigar inconsistência temporal antes de conclusão.
Grau atual insuficiente para condenação.
```

## Dicas e Armadilhas

### Dica
> Comece pela matriz de coerência. Ela revela rapidamente onde estão os conflitos no conjunto probatório.

### Dica 2
> Use a metáfora das palavras cruzadas literalmente: desenhe as "letras" que encaixam e as que não encaixam.

### Armadilha
> **"Uma evidência forte basta."**
>
> No foundherentismo, evidências isoladas são fracas.
> É o conjunto coerente que gera justificação forte.

### Armadilha 2
> **"Conflito = conjunto inválido."**
>
> Conflitos devem ser resolvidos, não ignorados.
> Explicar o conflito pode fortalecer a análise.
