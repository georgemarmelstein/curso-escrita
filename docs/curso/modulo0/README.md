# Módulo 0 - Preparação

> **Propósito:** Preparar o aluno para usar LLMs de forma eficiente e segura
> **Natureza:** Pré-requisito técnico
> **Interface Principal:** Claude Web

---

## Visão Geral

Este módulo prepara o terreno antes do conteúdo conceitual. Ao final, você terá:
- Conta configurada no Claude
- Privacidade ajustada conforme Resolução 615/2025
- Domínio básico da interface
- Memória e projetos configurados
- Noção de modo agêntico

---

## Objetivos do Módulo

1. Configurar corretamente a interface do LLM
2. Entender opções de privacidade e segurança
3. Dominar navegação básica e recursos principais
4. Configurar memória e personalização
5. Conhecer o modo agêntico básico

## Por que Este Módulo Importa

**Para profissionais do Direito:**
- A Resolução CNJ 615/2025 exige cuidados específicos com dados processuais
- Configuração errada pode violar sigilo profissional
- Interface mal utilizada desperdiça tempo e recursos

**Pré-requisito absoluto:**
- Sem este módulo, os conceitos dos Módulos 1-3 não podem ser praticados
- É como aprender a dirigir antes de estudar mecânica
