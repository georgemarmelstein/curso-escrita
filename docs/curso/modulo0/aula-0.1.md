# 0.1 Primeiros Passos

**Módulo:** 0 - Preparação
**Tempo estimado:** 15 minutos

---

## Objetivos

Ao final desta aula, você será capaz de:
- Criar uma conta no Claude
- Escolher o plano adequado ao seu perfil de uso
- Acessar a interface principal

---

## Por que Importa

**Para profissionais do Direito:**
- Claude é a base recomendada para este curso pela robustez em análise jurídica
- Escolha errada de plano pode limitar seu aprendizado ou gerar custos desnecessários
- Configuração inicial correta evita problemas futuros

**Contexto:**
Os conceitos do curso se aplicam a qualquer LLM (ChatGPT, Gemini), mas Claude oferece:
- Respostas mais estruturadas
- Melhor desempenho em textos longos
- Controle explícito de privacidade

---

## Base Conceitual

### Por que Claude como Plataforma Principal?

| Razão | Significado Prático |
|-------|---------------------|
| **Robustez cognitiva** | Melhor para análise jurídica, argumentação complexa, escrita técnica |
| **Transparência** | Anthropic explica funcionamento e dá controle sobre dados |
| **Didática** | Interface limpa, comportamento previsível |

### Os Planos Disponíveis

| Plano | Preço | Indicação | Uso no Direito |
|-------|-------|-----------|----------------|
| **Free** | R$ 0 | Curiosos, testes | Insuficiente para trabalho real |
| **Pro** | ~US$ 20/mês | Profissionais | **RECOMENDADO** |
| **Max** | US$ 100-200/mês | Alta demanda | Escritórios, uso intenso |

### Diferenças Práticas

**Free:**
- Limite baixo de mensagens
- Sem prioridade em horários de pico
- Sem extended thinking

**Pro (Recomendado):**
- 5x mais capacidade
- Acesso a todos os modelos (Opus, Sonnet)
- Extended thinking para análise profunda
- Prioridade de processamento

**Max:**
- 5x a 20x mais que Pro
- Para uso em escala ou equipes

## Exercício Prático

### Tarefa
Criar sua conta no Claude e escolher o plano adequado.

### Passo a Passo

**1. Acessar o site:**
- Navegador: Chrome, Firefox ou Edge (versões recentes)
- URL: **https://claude.ai**

**2. Criar conta:**
- Clique em "Sign up" ou "Criar conta"
- Escolha método:
  - Email + senha, OU
  - Conta Google, OU
  - Conta Apple
- Verifique o email

**3. Primeiro acesso:**
- Aceite termos de uso
- Você entra no Free automaticamente

**4. Upgrade para Pro (se aplicável):**
- Clique no perfil (canto inferior esquerdo)
- "Ver todos os planos"
- Selecione "Pro"
- Insira dados de pagamento
- Confirme

### Checklist de Verificação

```
[ ] Conta criada no claude.ai
[ ] Email verificado
[ ] Consegui acessar a interface de chat
[ ] (Se Pro) Assinatura confirmada
[ ] Entendo diferença entre Free, Pro e Max
```

---

## Dicas e Armadilhas

### Dica
> Comece no Free por alguns dias. Quando perceber que está "batendo no limite" frequentemente, faça upgrade. Assim você sente a diferença na prática.

### Armadilha
> **"Vou usar o Free porque não quero gastar."**
>
> Para uso profissional no Judiciário, o Free é inadequado:
> - Limites baixos interrompem trabalho
> - Sem prioridade = lentidão em horário comercial
> - Sem extended thinking = análises menos profundas
>
> O Pro custa menos que um livro jurídico por mês.
